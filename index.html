Privacy Policy for Job Fill Chrome Extension

Last updated: January 11, 2026

Job Fill respects your privacy. This extension is designed to help users autofill job application forms using information they voluntarily provide.

Data Collection

Job Fill does not collect, transmit, sell, or share personal data with third parties.

Data Storage

All user-provided information (such as name, contact details, education, and work history) is stored locally in the user’s browser using Chrome’s storage APIs. This data never leaves the user’s device.

Data Usage

Stored information is used solely to autofill job application forms on the currently active webpage when the user explicitly triggers the extension.

Permissions Requested

  • storage – to save user-provided data locally in the browser
  • activeTab – to access the currently active page when the user initiates autofill
  • scripting – to insert saved information into form fields on the active page

No other permissions are requested.

Third-Party Services

Job Fill does not use analytics services, advertising networks, or third-party trackers.

User Control

All autofill actions occur only after explicit user initiation. Users can edit or delete their stored data at any time by clearing extension storage or uninstalling the extension. The extension does not run automatically or in the background.

Data Retention and Deletion

User data is retained only within the browser’s local storage and is permanently removed when the user clears extension storage or uninstalls the extension.

Changes

This privacy policy may be updated if the extension’s functionality changes. Any updates will be reflected on this page.
